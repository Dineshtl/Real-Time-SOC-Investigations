## 👨‍💻 About This Repository

This repository contains **real-time SOC investigations** conducted on live Splunk ES alert data. Each investigation follows a structured SOC workflow:

**Detect → Triage → Investigate → Threat Intel → Verdict → Recommend**

All alerts were generated by **custom correlation rules** built in Splunk Enterprise Security on real FortiGate firewall traffic data.

## 🧠 Investigation Methodology

Each report covers:
- **Alert Details** — rule fired, urgency, security domain
- **Raw Event Analysis** — SPL queries, event count, timeframe
- **Attacker Profiling** — IP, ASN, country, subnet
- **Threat Intelligence** — VirusTotal + Cisco Talos verdict
- **MITRE ATT&CK Mapping** — tactic and technique identification
- **Verdict** — True Positive / False Positive with justification
- **Recommendations** — immediate, short-term, long-term actions
